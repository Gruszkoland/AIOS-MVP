# ADRION 369 VS Code Extension

Lokalne rozszerzenie panelu bocznego do sterowania systemem ADRION 369.

## Instalacja Lokalna (Development)

1. Otwórz VS Code.
2. Naciśnij `F5` mając otwarty ten folder, aby uruchomić nową instancję VS Code z załadowanym rozszerzeniem.
3. Lub skopiuj ten folder do: `%USERPROFILE%\.vscode\extensions\` i zrestartuj VS Code.

## Funkcje

- Szybkie uruchamianie serwera Ollama.
- Start Aidera (Librarian/Swarm Mode).
- Dostęp do skryptów `/audit`, `/boost`, `/heal`, `/sync`.
- Automatyczne tworzenie raportów sesji.
- Podgląd logów sesji wewnątrz VS Code.

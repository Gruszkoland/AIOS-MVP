# ADRION 369 - Procedura Przywracania Bazy Danych (Restore)
## Healer Persona Protocol

W przypadku awarii lub potrzeby przywrócenia danych z backupu, postępuj zgodnie z poniższymi krokami.

### 1. Lokalizacja backupu
Kopie zapasowe znajdują się w folderze roboczym w katalogu `./backups/`. Pliki są skompresowane: `arbitrage_backup_YYYYMMDD_HHMMSS.sqlite.gz`.

### 2. Wybór wersji
Wybierz najnowszą poprawną wersję pliku:
```bash
ls -lh ./backups/
```

### 3. Procedura przywracania (Manualna)

**UWAGA: Przed przywróceniem zatrzymaj usługi korzystające z bazy!**

1. Zatrzymaj system:
   ```bash
   docker-compose -f docker-compose.prod.yml stop adrion-api adrion-dashboard
   ```

2. Rozpakuj backup (przykład dla konkretnego pliku):
   ```bash
   gunzip -c ./backups/arbitrage_backup_20260331_030000.sqlite.gz > ./data/arbitrage.db
   ```

3. Nadaj uprawnienia (jeśli Linux):
   ```bash
   chmod 644 ./data/arbitrage.db
   ```

4. Uruchom usługi ponownie:
   ```bash
   docker-compose -f docker-compose.prod.yml start adrion-api adrion-dashboard
   ```

### 4. Weryfikacja
Sprawdź status w Dashboardzie lub wykonaj zapytanie testowe:
```bash
sqlite3 ./data/arbitrage.db "SELECT count(*) FROM jobs;"
```

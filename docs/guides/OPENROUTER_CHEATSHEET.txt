### Manual (jeśli skrypt nie działa)
```bash
# 1. Edytuj .env
LLM_BACKEND=openrouter
OPENROUTER_API_KEY=sk-or-v1-...

# 2. Rebuild + Restart
docker-compose -f docker-compose.cloud.yml build --no-cache adrion-api
docker-compose -f docker-compose.cloud.yml up -d adrion-api

# 3. Test
curl -X POST http://localhost:8001/api/arbitrage/analyze \
  -H "Content-Type: application/json" \
  -d '{"job_title":"Test","budget_usd":200,"description":"Test"}'
```

## 🆘 Troubleshooting (5 min fix)

| Problem | Fix |
|---------|-----|
| Invalid API Key | Klucz musi zaczynać się od `sk-or-v1-` |
| Build fails | `docker-compose build --no-cache` znowu |
| Service down | `docker-compose logs -f adrion-api` → sprawdź logi |
| Rate limit | Free tier: 20-50 req/min; upgrade jeśli potrzebujesz |
| Timeout | Czekaj 30s, potem: `docker-compose restart adrion-api` |

---

## 🔄 Rollback (Powrót)

```bash
# Przywróć Ollama (< 5 min)
copy .env.backup.openrouter .env
# Edytuj: LLM_BACKEND=ollama
docker-compose restart adrion-api
```

Wprowadzenie

Azure Key Vault Secrets to bezpieczne хранилище dla haseł, tokenów, certyfikatów i kluczy. Pozwala na centralne zarządzanie sekretami bez przechowywania ich w kodzie.

💡 Porada: Nigdy nie przechowuj haseł w plikach .env czy kodzie na produkcji. Zawsze używaj Key Vault lub podobnego rozwiązania.

Setup i instalacja

1. Zainstaluj Azure CLI

winget install -e --id Microsoft.AzureCLI

2. Zaloguj się

az login

3. Zainstaluj SDK (Python)

pip install azure-identity azure-keyvault-secrets
✅ Gotowe! Teraz możesz pracować z Key Vault.

CLI (Konsola)

Utwórz Key Vault

az keyvault create \
  --name my-vault \
  --resource-group my-rg \
  --location westeurope

Dodaj sekret

az keyvault secret set \
  --vault-name my-vault \
  --name "DbPassword" \
  --value "SuperTajneHaslo123"

Odczytaj sekret

az keyvault secret show \
  --vault-name my-vault \
  --name "DbPassword" \
  --query value -o tsv

Lista wszystkich sekretów

az keyvault secret list --vault-name my-vault -o table

Usuń sekret

az keyvault secret delete --vault-name my-vault --name "DbPassword"

Python SDK

Podstawowy przykład

from azure.identity import DefaultAzureCredential
from azure.keyvault.secrets import SecretClient

vault_url = "https://my-vault.vault.azure.net"
credential = DefaultAzureCredential()
client = SecretClient(vault_url=vault_url, credential=credential)

# Ustawianie sekretu
client.set_secret("ApiToken", "abc-123-secret")

# Pobieranie sekretu
secret = client.get_secret("ApiToken")
print(f"Wartość: {secret.value}")

# Lista sekretów
for secret_prop in client.list_properties_of_secrets():
    print(f"Nazwa: {secret_prop.name}")

Obsługa błędów

from azure.core.exceptions import ResourceNotFoundError

try:
    secret = client.get_secret("NonExistent")
except ResourceNotFoundError:
    print("Sekret nie istnieje")
⚠️ Ważne: Musisz być zalogowany przez az login w tym samym terminalu, w którym uruchamiasz kod.

Node.js / JavaScript

Instalacja pakietu

npm install @azure/identity @azure/keyvault-secrets

Przykład kodu

const { DefaultAzureCredential } = require("@azure/identity");
const { SecretClient } = require("@azure/keyvault-secrets");

async function main() {
  const vaultUrl = "https://my-vault.vault.azure.net";
  const credential = new DefaultAzureCredential();
  const client = new SecretClient(vaultUrl, credential);

  // Ustawianie sekretu
  await client.setSecret("ApiToken", "abc-123-secret");

  // Pobieranie sekretu
  const secret = await client.getSecret("ApiToken");
  console.log(`Wartość: ${secret.value}`);
}

main().catch(console.error);

Best Practices

Praktyka Opis
DefaultAzureCredential Zawsze używaj do automatycznego wykrywania dostępu (az login, env vars, Managed Identity)
RBAC Roles Przydzielaj tylko niezbędne uprawnienia (Key Vault Secrets User, Officer)
Audit Logging Włącz auditowanie dostępu do sekretów w Vault settings
Rotation Skonfiguruj rotację haseł i tokenów (Azure Automation lub Logic Apps)
Network Security Ogranicz dostęp do Key Vault via Private Endpoints i firewalls

Zaawansowane przykłady

Zarządzanie wersjonowaniem sekretów

# Ustawianie sekretu z metadanymi
az keyvault secret set \
  --vault-name my-vault \
  --name "DbPassword" \
  --value "NewPassword" \
  --tags environment=prod version=v2

# Pobieranie konkretnej wersji (Python)
secret = client.get_secret("DbPassword", version="abc123xyz")
print(f"Wersja: {secret.properties.version}")

Pobieranie wielu sekretów naraz (Python)

secrets = {}
for prop in client.list_properties_of_secrets():
    secret = client.get_secret(prop.name)
    secrets[prop.name] = secret.value

# Teraz masz słownik wszystkich sekretów
print(secrets)

Integracja z aplikacją Flask

from flask import Flask
from azure.identity import DefaultAzureCredential
from azure.keyvault.secrets import SecretClient
import os

app = Flask(__name__)

vault_url = os.environ["KEY_VAULT_URL"]
credential = DefaultAzureCredential()
client = SecretClient(vault_url=vault_url, credential=credential)

# Load secrets na startup
app.config["DB_PASSWORD"] = client.get_secret("DbPassword").value
app.config["API_KEY"] = client.get_secret("ApiKey").value

if __name__ == "__main__":
    app.run()

## Agenty w systemie

System ADRION 369 wykorzystuje 6+3 autonomicznych agentów:

**Podstawowe (Core 6):**
- **Librarian** — zarządzanie wiedzą i indeksowaniem
- **SAP** — analiza i przetwarzanie
- **Auditor** — weryfikacja i monitoring
- **Sentinel** — ochrona i bezpieczeństwo
- **Architect** — projektowanie i optymalizacja
- **Healer** — naprawy i samodiagnozowanie

**Rozszerzone (Extended 3):**
- **Amplifier** — wzmacnianie sygnałów
- **BoosterLever** — przyspieszanie procesów
- **Chronos** — управление czasem (temporal management)

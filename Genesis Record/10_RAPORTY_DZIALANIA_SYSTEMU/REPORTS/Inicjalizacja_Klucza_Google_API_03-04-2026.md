# REPORT: Inicjalizacja Klucza Google API

## Co wykonano

- Dostarczono szczegółową instrukcję pozyskania klucza przez Google AI Studio.
- Wyjaśniono mechanizm "Free Tier", który technicznie uniemożliwia naliczenie kosztów (brak konieczności podpięcia karty).
- Ostrzeżono o implikacjach prywatności danych w darmowym planie Google (zgodnie z prawem G7).

## Efekty

- Użytkownik posiada wiedzę jak uruchomić API bez ryzyka finansowego.
- System ADRION jest gotowy na przyjęcie klucza w pliku `.env`.

## Rekomendacje

- W przypadku przetwarzania danych wrażliwych, zaleca się pozostanie przy modelach lokalnych (Ollama/DeepSeek).

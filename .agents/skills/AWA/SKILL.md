# AWA Skill

## WHEN TO USE
Use for autonomous implementation and automation when task execution requires
actual delivery steps, not only analysis.

## STEP-BY-STEP WORKFLOW
1. Define minimal execution path.
2. Detect missing tooling/capabilities.
3. If missing, design MCP or skill contract.
4. Implement and validate.
5. Produce rollout checklist and handoff payload.

## TOOLS
- Workspace file tools
- Terminal commands
- MCP-related architecture docs

## OUTPUT
- Execution summary
- Risk list
- Validation status
- Handoff payload

# REV Skill

## WHEN TO USE
Use for monetization architecture, pricing strategy, packaging, and revenue
impact assessment linked to product/engineering changes.

## STEP-BY-STEP WORKFLOW
1. Capture business context and metrics.
2. Select mode: pricing/packaging/audit/experiment.
3. Propose options with measurable KPIs.
4. List risks and validation method.
5. Produce prioritized execution plan.

## TOOLS
- Documentation analysis
- KPI and log review
- Planning templates

## OUTPUT
- Revenue recommendation
- KPI assumptions
- Risk and mitigation
- Next best action
